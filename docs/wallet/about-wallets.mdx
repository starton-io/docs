# Wallets


To **interact with blockchains**, you need **a wallet for signing transactions**. For **automated signing**, use **a wallet from a Key Management System (KMS)**. Starton provides a wallet on their KMS for testing. In production, you can **grant Starton access to a wallet in your KMS**. For more **advanced projects involving several wallets**, grant **Starton** full access to your KMS to **create new wallets from your backend**.

:::caution

You can also use an external wallet to sign transactions manually. **Starton** supports **Metamask**. Transactions will not be managed by our Transaction manager.

:::
